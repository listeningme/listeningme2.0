# listening 福利請聽 2.0

一個成功的社會並不是看富裕的人過得多優渥，而是弱勢的如何自在的生存。


## Install

1. `npm install -g youmeb-cli`
2. `npm i`
3. `bower i`

## Run

If you are in develope mode: `npm start` to start page server and open a new client window to type `youmeb` to run youmebjs server 

## publish Page code
1. `gulp publish`
